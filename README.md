JRPG (пошаговая rpg) система со следующим цикличным игровым циклом:
- Бой, в котором игрок сражается с одним врагом, выбирая действия.
- Магазин, в котором игрок тратит валюту полученную после боя на получение предметов, дающих новые действия и уникальные эффекты.
- Игра продолжается до поражения игрока.
